Image Types

JPEG (Joint Photographic Experts Group)

A JPEG is best for photographic image and provides a compressed format that gives a small file size when faced with lots of color pixel variation

GIF (Graphic interchange format)

A Gif is best for limited colors. Allos for threshold transparency and animation.

PNG (Portable Network Graphics

A PNG does well with limited color and photographic content. It allows for gradient and subtle transparency.
